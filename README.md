### Host src/client files:

- ```npm install```
- ```npm run dev```

### Start src/server/server.js:

- ```cd ./src/server```
- ```npm install```
- ```cd ../..```
- ```npm run server```

### Build for production:

- ```npm run build```
- ```host your build directory somewhere...```
